A `ReactiveList` is a reactive wrapper around a `List` that copies on write.

When you modify any element on the list, the `ReactiveList` will automatically notify its observers.

```dart
final items = ReactiveList([1, 2]);

Effect(() {
  print('Items: ${items.value}');
});

items.add(3); // prints "Items: [1, 2, 3]"
items[0] = 10; // prints "Items: [10, 2, 3]"
```
