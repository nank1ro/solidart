A `SetSignal` is a reactive wrapper around a `Set` that copies on write.

When you modify any element on the set, the `SetSignal` will automatically notify its observers.

```dart
final items = SetSignal({1, 2});

Effect(() {
  print('Items: ${items.value}');
});

items.add(3); // prints "Items: {1, 2, 3}"
items.remove(1); // prints "Items: {2, 3}"
```
