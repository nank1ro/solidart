# ReadSignal API docs

A read-only [Signal].

## Constructor

```dart
ReadSignal(
  T initialValue, {
  SignalOptions<T>? options,
});
```

A read-only [Signal].
`initialValue` is the initial value of the signal.
`options` are the options of the signal.

See [SignalOptions](/api-docs/signal-options) for more details.
